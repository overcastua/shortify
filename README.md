# Pushniak Dmytro KV-12mp
## Shortify - URL shortening service

## Installation

```bash
$ yarn install
```

## Running the app

Rename <b>example.env</b> to <b>.env</b>;

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```


## Database

```bash
# start
$ docker-compose up -d

# stop
$ docker-compose down
```
