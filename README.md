# Shortify - URL shortening service

## Installation

```bash
$ yarn install
```

## Running the app

Rename <b>example.env</b> to <b>.env</b>;

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```


## Database

```bash
# start
$ docker-compose up -d

# stop
$ docker-compose down
```

## Documentation

Go to {host}:{port}/docs. 

Also, there is a json swagger file with exported docs.
